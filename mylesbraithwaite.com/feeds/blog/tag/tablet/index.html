You can view/edit Excel, Word, and PowerPoint documents (though a third party application).</p> <p>It looks like the fact that it doesn't include an email, calendar, contacts, and BBM is for security. But how many times are you somewhere without your phone?</p> <p>It has a front and rear facing camera so you can do some video conferencing. Which would most likely be proprietary to the BlackBerry. Hopefully Skype will come out with something soon.</p> <p>If you are going to buy it I would suggest getting some type of Cloud Storage option (like Dropbox) so you don't have to mess around with syncing files (PDF, DOC, Excel, etc.).</p> <ul> <li>Smaller than my iPad (could fit in your jacket pocket).</li> <li>Wirelessly connects to your BlackBerry for Email, Calendar, Contacts, and BBM.</li> <li>Download books using Kobo (the Indigo book store). Amazon has a Kindle app for BlackBerry phones so they also should come out with one for the PlayBook.</li> <li>Has access to Documents To Go, so you can edit Excel, Word, and PowerPoint files (mostly like will cost money ~$20).</li> <li>Has a built in PDF reader.</li> <li>It has both a front and rear facing camera. So you could do video calling off it (Skype has an application for BlackBerry phones so the might have one for the PlayBook).</li> <li>The web browser is based off WebKit (which powers Apple's Safari and Google's Chrome web browsers) so it should be standard compliant (but web apps like the CRM don't work so while) (Facebook and the like work fine).</li> <li>Has <strong>Mobile</strong> Flash, which isn't the full flash so I am not sure what may or may not work.</li> <li>It based off the QNX OS which is a variant of Unix for real-time tasks on an embedded system.</li> <li>The BlackBerry App Store looks okay. Most apps look more oriented towards the Enterprise user.</li> <li>Google Talk application so you can connect to Pidgin.</li> <li>Evernote - a great note taking and searching application.</li> <li>I couldn't find a remote desktop client (ex when you are in a meeting you could connect to your desktop).</li> <li>It has support to run Android applications but most Android applications don't run while on Android phones so if I had to guess it would be wonky support at best.</li> <li>Has support for an HDMI-out so you might be able to load a PowerPoint presentation on it and then display it on a projector/television (one that has HDMI support).</li> <li>It has an application similar to iTunes where you could download music, movies and podcasts, but I wouldn't buy anything off it.</li> <li>From what I can tell from the screen shots it looks like the calendar, contacts, and email is pretty much the same design as the phone version.</li> </ul>me@mylesbraithwaite.com (Myles Braithwaite)Thu, 07 Apr 2011 12:08:00 -0400http://mylesbraithwaite.com/blog/2011/apr/7/blackberry-playbook/RIMBlackBerry PlayBookTabletSome Thoughts on the iPadhttp://mylesbraithwaite.com/blog/2010/mar/22/ipad/<p><strong>This is a blog post I wrote the a few weeks after iPad was shown to the world and never got around to publishing it on this blog.
