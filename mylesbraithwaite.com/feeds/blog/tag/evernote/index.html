Next just type in <code>wine ~/Downloads/Evernote_3.1.0.1225.exe</code> and follow the normal installation process.</p> <div class="inline illustration"> <a href="http://media.mylesbraithwaite.com/attachments/blog_post/7/evernote-windows-client-in-ubuntu.png" title="Evernote in Ubuntu 9.04"><img alt="Evernote in Ubuntu 9.04" src="http://media.mylesbraithwaite.com/attachments/blog_post/7/evernote-windows-client-in-ubuntu.png" width="450"/></a> </div> <p>Evernote is really cool, but as you can see from the screen shot above, it doesn't look so good.
