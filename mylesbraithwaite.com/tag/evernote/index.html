Installing Evernote on Ubuntu 9.04
Written on 20th October, 2009 in Evernote, Ubuntu, Wine, and Linux.

<p><a href="http://www.evernote.com/">Evernote</a> is a note taking service that has clients for <a href="http://www.evernote.com/about/download/windows.php">Windows</a>, <a href="http://www.evernote.com/about/download/mac.php">Mac <span class="caps">OS</span> X</a>, <a href="http://www.evernote.com/about/download/iphone/">iPhone</a>, and some other systems I don&#8217;t use. Which is great but I doesn&#8217;t support&nbsp;Linux.</p> <p>The installation process is really simple just <a href="https://help.ubuntu.com/community/Wine" title="Ubuntu Community Documentation for Wine">install Wine</a> and download <a href="http://www.evernote.com/about/download/windows.php">Evernote 3.1 for Windows</a> (not the beta). Next just type in <code>wine ~/Downloads/Evernote_3.1.0.1225.exe</code> and follow the normal installation&nbsp;process.</p> <div class="inline illustration"> <a href="/media/cache/2d/85/2d85678f7a4b6f3341082b28f35b6ac2.jpg" title="Evernote in Ubuntu 9.04"><img src="/media/cache/2d/85/2d85678f7a4b6f3341082b28f35b6ac2.jpg" alt="Evernote in Ubuntu 9.04" width="450"></a> </div> <p>Evernote is really cool, but as you can see from the screen shot above, it doesn&#8217;t look so good. But I am impressed though that Wine added the Tray icon to my Gnome notification&nbsp;area.</p>
