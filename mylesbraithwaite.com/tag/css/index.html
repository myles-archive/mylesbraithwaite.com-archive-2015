New Website Design
Written on 27th April, 2011 in CSS, Website, Design, and Meta.

<p>Ever since I read <a href="http://www.alistapart.com/articles/responsive-web-design/">Responsive Web Design</a> by Ethan Marcotte I wanted to make my website work on browser sizes between 1000 to 300 pixel. I had something which was essentially a hack up for a long time but that was pretty much rubbish.
